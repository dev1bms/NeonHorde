# App Store Connect — Submission Answers (Neon Horde 1.0.0)

Honest answers for every ASC form, per GOAL §6 Phase 10. Enter these in the
web UI where fastlane can't set them.

## Age rating (2025+ questionnaire — tiers 4+/9+/13+/16+/18+)
- Violence: **Infrequent/Mild Cartoon or Fantasy Violence** (abstract neon
  geometric shapes shooting other shapes; no blood, no humans, no gore)
- Realistic violence / graphic content: **None**
- Horror/fear themes: **None**
- Mature/suggestive themes, profanity, drugs, gambling: **None**
- In-app controls (chat, UGC, sharing to other users): **None** (share feature
  uses the iOS share sheet only — no in-app social features)
- Unrestricted web access: **No**
- Medical/wellness content: **No**
- **Expected rating: 9+** (13+ acceptable if violence judged "frequent")

## App privacy
- Data collection: **"No, we do not collect data from this app"** →
  privacy label **Data Not Collected**
- Rationale: no accounts, no network calls, no analytics/ads SDKs; saves are
  on-device only; PrivacyInfo.xcprivacy in the binary declares UserDefaults
  (CA92.1) + file-timestamp (C617.1) required-reason APIs, zero collected data.

## EU Digital Services Act
- Trader status: **Non-trader** (free hobby app, no monetization). EU users
  will see the standard non-trader consumer notice.

## Pricing & availability
- Price: **Free (Tier 0)** — set explicitly.
- Territories: **All**.

## Export compliance
- `ITSAppUsesNonExemptEncryption = NO` is in the Info.plist — no per-build
  compliance prompt. Uses only OS-provided TLS-free... (no networking at all).

## App Review information
- Sign-in required: **No**
- Demo account: **N/A**
- Contact: values from `~/.appstoreconnect/neonhorde.env`
  (REVIEW_FIRST_NAME / REVIEW_LAST_NAME / REVIEW_PHONE / REVIEW_EMAIL)
- Notes for reviewer:
  > Fully offline single-player arcade game. No login, no UGC, no ads, no
  > IAP, no data collection. Abstract geometric combat only. All art and
  > audio are procedurally generated by the developer's own code.

## Accessibility Nutrition Labels (optional in mid-2026)
- Defer for 1.0; revisit when mandatory. Game is fully playable muted
  (visual + haptic feedback carry all information).
